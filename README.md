# phase-1Final-Project
This is a simple web application for displaying news articles. It allows users to retrieve news articles from two categories using the News API.

# Features
display general news articles.
display sports news articles.
Search for news articles based on a search.
View details of each news article, including the title, description, and a link to the full article.
# Project mvps 
display latest new headlines 
show at least two different news categories through filters 
create a feature that allows a user to search for news with keywords
